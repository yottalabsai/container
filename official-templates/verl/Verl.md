# Verl

Verl is a framework for **LLM post-training and reinforcement learning (RL)**. Typical use cases include:
- Engineering training workflows such as RLHF, DPO, and PPO
- Distributed training, data and sampling pipelines, and training job orchestration

> This template includes a small model inference demo (Qwen/Qwen2.5-0.5B-Instruct) to verify the image works out of the box.
> The inference demo does not represent Verl's core capabilities.

## Files (inside the image)

- `/workspace/demo_qwen_infer.py`: Pre-built inference demo (console output)
- `/workspace/Verl_Qwen2.5_Demo.ipynb`: Notebook demo
- `/workspace/EXAMPLES.md`: Installation and usage examples
